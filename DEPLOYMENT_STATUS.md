## 🔧 故障排除

### 网站还是显示旧版本？

**方法1：清除浏览器缓存**
```
Chrome/Edge: Cmd+Shift+Delete (Mac) / Ctrl+Shift+Delete (Win)
Safari: Cmd+Option+E
Firefox: Cmd+Shift+Delete
```

**方法2：使用隐私模式**
```
Chrome: Cmd+Shift+N
Safari: Cmd+Shift+N  
Firefox: Cmd+Shift+P
```

**方法3：检查Netlify部署状态**
```
https://app.netlify.com
→ time-2025站点
→ Deploys标签
→ 查看最新部署是否成功
```

### 图表不显示？

**检查：**
1. 浏览器控制台是否有错误（F12）
2. ECharts CDN是否加载成功
3. 网络连接是否正常

**解决：**
1. 刷新页面
2. 检查网络
3. 尝试不同浏览器
