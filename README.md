# 🍅 番茄钟 Pomodoro Timer

一个简洁优雅的番茄钟应用，帮助您专注工作，高效管理时间。

## ✨ 特性

- ⏱️ **经典番茄钟** - 25分钟工作，5分钟短休息，15分钟长休息
- 📊 **数据统计** - 记录每日番茄数和专注时长
- 🎨 **美观界面** - 现代化设计，支持亮色/暗色主题
- 📱 **响应式** - 完美适配桌面端和移动端
- 🔔 **通知提醒** - 计时结束时的友好提醒
- 💾 **本地存储** - 数据保存在浏览器本地
