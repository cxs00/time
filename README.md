# 🍅 番茄时钟 Web 应用

一个功能完整、界面美观的番茄时钟（Pomodoro Timer）Web应用，帮助你提高工作效率和时间管理能力。

![番茄时钟](https://img.shields.io/badge/version-1.0-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ 功能特性

### 核心功能
- ⏱️ **番茄时钟计时** - 25分钟工作，5分钟短休息，15分钟长休息
- 🎯 **自动循环** - 完成4个番茄后自动进入长休息
- ⏸️ **暂停/继续** - 随时暂停和继续计时
- 🔄 **重置/跳过** - 灵活控制时钟状态
- 📊 **进度可视化** - 圆形进度条实时显示剩余时间

### 数据统计
- 📅 **今日统计** - 查看今天完成的番茄数和工作时长
- 📈 **本周统计** - 柱状图展示本周每日完成情况
- 🏆 **历史记录** - 总番茄数、使用天数、连续天数
- 📝 **最近记录** - 查看最近的工作和休息记录

### 个性化设置
- ⏱️ **自定义时长** - 自由设置工作、休息时间
- 🔔 **通知提醒** - 浏览器通知 + 音效提醒
- 🎵 **音效控制** - 可开关音效反馈
- 🎨 **主题切换** - 浅色/深色/跟随系统
- 🤖 **自动开始** - 可设置自动开始下一阶段

### 数据管理
- 💾 **本地存储** - 所有数据保存在本地，无需网络
- 📤 **数据导出** - 导出统计数据和设置
- 🗑️ **数据清除** - 一键清除所有数据

## 🚀 快速开始

### 在线使用
直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何依赖。

### 本地部署
```bash
# 克隆或下载项目
cd pomodoro-timer

# 使用任意Web服务器打开
# 方式1：使用Python
python -m http.server 8000

# 方式2：使用Node.js
npx serve

# 方式3：直接双击 index.html
```

然后在浏览器中访问：`http://localhost:8000`

## 🎯 使用方法

### 基本操作
1. **开始工作** - 点击"开始"按钮启动25分钟工作计时
2. **暂停/继续** - 随时暂停或继续当前计时
3. **重置** - 重置当前计时器到初始状态
4. **跳过** - 跳过当前阶段，进入下一个阶段

### 快捷键
- `空格键` - 开始/暂停
- `R` - 重置
- `S` - 跳过

### 模式切换
点击顶部的"工作"、"短休息"、"长休息"标签可以手动切换模式（仅在计时器停止时可用）。

### 查看统计
点击右上角的"📊 统计"按钮查看详细的使用统计数据。

### 自定义设置
点击右上角的"⚙️ 设置"按钮进入设置页面，可以：
- 调整工作和休息时长
- 设置长休息间隔
- 启用/关闭通知和音效
- 切换主题模式
- 导出或清除数据

## 🎓 番茄工作法介绍

番茄工作法是由Francesco Cirillo在1980年代末发明的时间管理方法：

1. **选择任务** - 确定要完成的任务
2. **设定番茄时钟** - 通常为25分钟
3. **专注工作** - 直到番茄时钟响起
4. **短暂休息** - 休息3-5分钟
5. **重复** - 每4个番茄时钟后，休息15-30分钟
