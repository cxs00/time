## 📝 使用方法

### 安装系统
```bash
cd ~/Desktop/通用部署
./install.sh
```

### 创建项目
```bash
# 使用增强版生成器
./create-new-project-enhanced.sh

# 或使用图形界面
./ui/cli-interface.sh
```

### 系统管理
```bash
# 安全检查
./security/encryption.sh security_audit

# 查看系统状态
./ui/cli-interface.sh

# 生成系统报告
./monitoring/logger.sh generate_system_report
```
