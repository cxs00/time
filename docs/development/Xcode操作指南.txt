✅ 所有文件已准备就绪！
✅ Xcode已打开！

现在请在Xcode中完成以下操作：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📁 步骤1：添加Web文件夹 (最重要！)

1. 在Xcode左侧项目导航器中找到 "time" 文件夹
2. 右键点击 "time" 文件夹
3. 选择 "Add Files to "time"..."
4. 在弹出的文件选择器中：
   📍 导航到：time/time/Web
   📂 选择 "Web" 文件夹
   
   ⚠️  关键设置：
   ┌─────────────────────────────────────────────┐
   │ ☑ Copy items if needed                      │
   │ ☑ Create folder references  👈 选这个！      │
   │   (蓝色文件夹图标，不是黄色)                  │
   │                                             │
   │ Add to targets:                             │
   │ ☑ time                                      │
   └─────────────────────────────────────────────┘
   
5. 点击 "Add" 按钮

✓ 成功标志：看到蓝色的 "Web" 文件夹出现在项目中

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 步骤2：添加Swift文件

1. 右键点击 "time" 文件夹
2. 选择 "Add Files to "time"..."
3. 选择文件：time/time/PomodoroWebView.swift
4. 确保勾选 "Add to targets: time"
5. 点击 "Add"

✓ 成功标志：看到 "PomodoroWebView.swift" 出现在文件列表

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶️  步骤3：运行App

1. 在Xcode顶部工具栏，点击设备选择器
2. 选择一个iPhone模拟器（如 iPhone 15 Pro）
3. 点击 ▶️ 播放按钮（或按 Cmd + R）
4. 等待编译...
5. 🎉 番茄时钟App启动！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🐛 如果遇到问题：

问题1：编译错误 "Cannot find 'PomodoroWebView'"
解决：确认步骤2中的Swift文件已正确添加

问题2：WebView显示空白
解决：确认步骤1中选择了 "Create folder references"（蓝色文件夹）

问题3：找不到index.html
解决：检查Web文件夹在项目中是蓝色的，不是黄色的

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 小贴士：

• 蓝色文件夹 = folder references（正确）
• 黄色文件夹 = groups（错误）

• 如果添加错了，删除后重新添加
• Clean Build: Product → Clean Build Folder (Cmd + Shift + K)
